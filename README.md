Title: Modelling implicit bias: linking language, the media, and biased attitudes

Principal Investigator: Dermot Lynott  Data Manager:Alexander Porshnev

Affiliation: Maynooth University

Funder: Research Ireland Grant number: 21/FFP-P/10118 Start date: 23-09-2023  End date: 23-10-2026

Project abstract: 
The central thesis of this proposal is that biased and prejudicial attitudes, common in our society, are encoded in the way we use language. These biases can then be uncovered using appropriate statistical and machine learning models, and used to model and predict human behaviour. The proposers have an excellent track record in the areas of cognitive modelling, machine learning, artificial intelligence and behavioural experimentation, and this proposal will allow them to address this fundamental issue in a novel, and rigorous fashion.
